A clean and responsive To-Do List web application built using Tailwind CSS and JavaScript.  
This app allows users to add, delete, and clear tasks with data stored in the browser using LocalStorage.  

1. Built with beginner-friendly code  
2. Uses Tailwind CDN for styling  
3. Saves tasks even after page refresh  
4. Works fully in the browser

Technologies Used:
- HTML  
- Tailwind CSS  
- JavaScript (DOM + LocalStorage)
